## 决策项 1：T3 硬件原型预算方向

### 选项 A · 低端路线：RK3588 + RK1828 开发板

| 项 | 参数 |
|---|---|
| BOM | ~$300-500 |
| 目标 | RK1828 3D-DRAM 外置协处理器实测 |
| 数据价值 | ★★★★★ 直接验证 DTA 低端形态 |
| 风险 | 低（有官方 SDK） |
| 时间 | 3-4 周（等货 + 熟悉 SDK） |

**产出**：Qwen3-8B 在窄接口（PCIe 2.0 x1 / USB 3.0）外置协处理器上的**端到端延迟 + tokens/s**。补齐 v0.5 只有 M4 内嵌数据的空缺。

### 选项 B · 高端路线：DGX Spark

| 项 | 参数 |
|---|---|
| BOM | $4,699（涨价后） |
| 目标 | 200GbE ConnectX-7 上的 LLM 分布式推理 |
| 数据价值 | ★★★ 是 DTA 一体化形态的参考实现 |
| 风险 | 中（供应紧张，等货可能 3+ 个月） |
| 时间 | 6-12 周 |

**产出**：LPDDR5X 273 GB/s 内部 vs 25 GB/s 外部的实测数据。但**这更多是"验证已有产品"**，不是"验证 DTA 主张"。

### 大聪明的建议 → **选项 A**

**理由**：
1. **数据价值/成本比更高**：$500 换一份"8B 模型跑在窄接口外置协处理器"的直接实测 vs. $4,699 换一份"NVIDIA 官方参考实现"的复述数据
2. **补齐关键空白**：v0.5 只有 M4 unified memory 的数据（内嵌案例），缺"真·外置协处理器"的端到端实测。RK1828 补上这块
3. **风险可控**：Rockchip 生态稳定，RKNN3 SDK 已经放出 Qwen3 8B 61 TPS 的官方数据，我们做的是**独立复现 + 加测外部延迟**
4. **供应链风险**：DGX Spark 涨价 18% 本身就是 v0.5 观察 7 的证据——**现在花 $4,699 买它有点"用高价证明高价合理"的循环论证味道**

**如果预算允许**：先跑 A（$500，3-4 周），出真数据后再决定要不要上 B。
